# Writing style — standing instruction

**Do not write in punchy "marketing/infomercial" register.** No short, dramatic, staccato sentences played for effect. Write like a human analyst: measured, flowing prose in full sentences and proper paragraphs.

Specifically avoid:
- Hype fragments and one-line drama ("The jailbreak was real and public.", "Plan accordingly.", "Here's the thing.")
- "X, not Y" headlines and antithesis ("A category change, not a news cycle")
- Rule-of-three escalation ("It didn't restrict a chip. It didn't sanction a company. It reached in and...")
- Arrow / "from X to Y" constructions
- Faux-insight subtitles ("the mechanism matters more than the headline")
- Em-dash overuse and excessive bolding

Default register for all deliverables (SITREP, bulletins, PDFs, web copy, LinkedIn): neutral, factual, declarative, in connected prose. Use the `detect_ai_tells` check before shipping and fix anything it flags plus any register slop it misses.
